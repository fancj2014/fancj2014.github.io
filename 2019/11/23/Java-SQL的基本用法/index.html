导入sqlite-jdbc

首先从sqlite-jdbc库下载sqlite-jdbc-(VERSION).jar的最新版本。

然后需要将.jar包加入到你的eclipse工程中：右键你的工程->Build Path->configure Build Path->Libraries->Add External JARs->找到sqlite-jdbc-version.jar包导入就可以了。
